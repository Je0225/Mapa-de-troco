# Calculador de Troco 

Aplicação que simula o caixa de um estabelecimento. O usuário informa o valor do troco e a quantidade de cada cédula e moeda disponível no caixa. O sistema calcula e exibe a combinação ideal de cédulas e moedas necessárias para devolver o troco corretamente.

## Funcionalidades

![Captura de tela](tela_principal.png)

- Aplicação Windows
- Recebe o valor do troco a ser devolvido.  
- Permite informar a quantidade de cada cédula e moeda disponível.  
- O usuário pode escolher se o sistema deve fazer a combinação com poucas ou muitas unidades.
- Calcula automaticamente a melhor combinação possível de notas e moedas.  
- Exibe o resultado de forma clara e organizada.

## Tecnologias
  - C#
  - Windows Forms
  - .NET Framework 4.7.2
